- Опыт работы с Rust/С

- Глубокое понимание принципов работы сетей

#### 2. Java Developer
**Требования:**
- Опыт разработки на Java
- Знание Spigot/Paper API
- Опыт работы PostgreSQL
- **Будет плюсом:** опыт работы с RabbitMQ/Kafka

#### 3. Golang Developer (Network)
**Требования:**
- Опыт разработки на Go
- Глубокое понимание сетевых протоколов (TCP/UDP, HTTP/2)
- Опыт работы с Kafka, Redis / Valkey, ClickHouse
- Знание Kubernetes и containerization
- **Будет плюсом:** опыт с gRPC, NATS/Kafka, распределёнными системами

#### 4. Python ML Developer
**Требования:**
- Опыт разработки на Python
- Знание библиотек машинного обучения (scikit-learn, PyTorch / TensorFlow)
- Опыт обработки и анализа данных (Pandas, NumPy)
- Понимание алгоритмов ML и методов оптимизации
- **Будет плюсом:** опыт развёртывания моделей в продакшн (MLflow, ONNX, Docker)
